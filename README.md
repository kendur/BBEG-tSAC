# TSAC Universe Repository

This repository contains the canonical lore, cosmology, and system design for:

**The Silent Accord Chronicles (TSAC)**

TSAC is a transmedia world exploring the balance between ideological planes of existence.

The universe is shaped by two metaphysical forces:

• **Brighthaven** — light, certainty, justice  
• **Shadewoven** — shadow, reflection, doubt  

A betrayal within the order of cosmic guardians corrupted the balance between these planes, destabilizing the structure of reality.

This repository serves as the **source of truth** for the TSAC universe.

# Project Philosophy

TSAC explores themes such as:

• balance vs domination  
• certainty vs reflection  
• redemption vs judgment  
• suffering as corruption  

The goal is to create a durable universe capable of supporting:

• novels  
• tabletop games  
• digital games  
• AI-driven storytelling
