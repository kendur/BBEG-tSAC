## How to Use This File

When an idea is generated (in session or independently) but is not ready or suitable for canon, note it here. Include enough context to understand the idea later and a brief note on why it wasn't used.

If an idea here becomes relevant, move it to `SCRATCHPAD/raw_conversation_extractions.md` for processing, or directly promote it to the appropriate canon or story file if it's already well-developed.
