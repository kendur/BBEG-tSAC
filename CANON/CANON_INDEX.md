# TSAC Canon Index

This document provides a structured overview of the canonical lore for the **Silent Accord Chronicles (tSAC)** universe.

The TSAC setting is governed by a cosmic equilibrium system known as the **[Concordance](cosmology/concordance.md)**, which maintains balance between two fundamental forces:

- **[Brighthaven](cosmology/energies/brighthaven.md)** — order, structure, preservation  
- **[Shadewoven](cosmology/energies/shadewoven.md)** — entropy, transformation, dissolution  

All major events within the setting ultimately trace back to disturbances within this balance.
